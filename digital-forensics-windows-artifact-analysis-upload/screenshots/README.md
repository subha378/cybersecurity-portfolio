# Screenshot Redaction Notes

These screenshots were created from a simulated digital forensics report and redacted for public portfolio use.

## Redacted items
- Usernames and account identifiers
- Email addresses
- Device IDs and serial values
- Search queries and full URLs
- File paths and filenames that may identify the case/user
- Table rows with timestamps or private evidence values
- Preview/details panes containing raw case information

## Purpose
The goal is to provide visual proof of Autopsy tool usage and investigation workflow while protecting privacy and academic integrity.
