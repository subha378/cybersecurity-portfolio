# Applied Cryptography Project — Corrected Portfolio Version

## Overview
This is a cleaned, portfolio-safe version of an applied cryptography project covering RSA, ElGamal secure voting, SHA-256 proof-of-work, AES/session key concepts, and encryption ethics.

The original university assessment files are **not included**. This public version includes corrected explanations, safe demonstration code, and recruiter-friendly documentation without student identifiers, raw assignment answers, plaintext/ciphertext files, private keys, or university-restricted material.

## Why this version exists
The assessed submission received a strong mark overall, but some parts had small technical issues. This portfolio version keeps the useful project evidence while correcting the concepts that needed improvement.
