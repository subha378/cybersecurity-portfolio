# Project Summary — Applied Cryptography

## Purpose
This project shows practical understanding of encryption, hashing, public-key cryptography and blockchain-style proof-of-work using safe demonstration code.

## What was cleaned for public release
The original university files were not uploaded because they may contain assessment material, student identifiers and provided data. Instead, this portfolio version contains:
- A cleaned README
- Safe demonstration code
- A summary of concepts learned
- No original assignment notebooks
- No plaintext/ciphertext files from assessment
- No student ID or private data

## Key Learning Outcomes
- RSA demonstrates asymmetric encryption using public/private keys.
- ElGamal demonstrates public-key encryption and introduces privacy-preserving voting concepts.
- AES demonstrates symmetric encryption with a shared secret key.
- SHA-256 proof-of-work demonstrates hash difficulty and nonce searching.
- Modular arithmetic is foundational to many cryptographic systems.

## Recruiter-Friendly Summary
This project demonstrates that I can understand and explain core cryptographic systems, write clear technical documentation, and use Python to model security concepts safely.
