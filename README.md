## Portfolio Projects

### 1. Digital Forensics — Windows Artifact Analysis

A simulated digital forensics case study using **Autopsy** to analyse Windows artifacts. The project involved reviewing evidence source details, OS accounts, USB device history, web history, installed programs, video file artifacts, evidence integrity and PII handling.

**Tools:** Autopsy
**Skills:** Digital Forensics, Windows Artifact Analysis, Chain of Custody, USB Artifact Review, Web History Analysis, Installed Program Analysis, Technical Reporting, PII Handling

**Folder:** `digital-forensics-windows-artifact-analysis-upload`

---

### 2. Cryptography Fundamentals — Symmetric Encryption & Hashing

A practical cryptography project covering classical ciphers, AES symmetric encryption, Fernet encryption, hashing, birthday attacks, second preimage concepts and SHA3-256 avalanche effect analysis.

**Tools:** Python, Jupyter Notebook, cryptography library
**Skills:** AES-CBC, AES-CTR, Fernet, Hashing, SHA3-256, Birthday Attack, Cryptanalysis, Technical Documentation

**Folder:** `02-applied-cryptography`

---

### 3. Applied Cryptography — RSA, ElGamal & Secure Voting

A corrected, portfolio-safe applied cryptography project covering RSA key exchange concepts, ElGamal secure voting, homomorphic aggregation, SHA-256 proof-of-work and encryption ethics.

**Tools:** Python, PARI/GP concepts, Jupyter Notebook concepts
**Skills:** RSA, ElGamal, Secure Voting, Public-Key Cryptography, SHA-256, Proof-of-Work, Modular Arithmetic, Encryption Ethics

**Folder:** `applied-cryptography-corrected-safe-portfolio`
